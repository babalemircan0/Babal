# Babal - ASP.NET Core MVC Projesi

Bu uygulama, kullanıcıların kayıt olabileceği ve giriş yapabileceği temel bir üyelik sistemidir.

## 🛠 Kullanılan Teknolojiler
- **Backend:** ASP.NET Core 8.0 MVC
- **Veritabanı:** SQLite (Entity Framework Core)
- **Oturum Yönetimi:** Session
- **Frontend:** HTML5, CSS3, Bootstrap

## 🚀 Çalıştırma Talimatları
1. Projeyi bilgisayarınıza indirin.
2. Visual Studio ile açın.
3. `bin` ve `obj` klasörlerini temizleyip projeyi derleyin.
4. Uygulamayı başlattığınızda otomatik olarak `/Account/Register` sayfasına yönlendirileceksiniz.
